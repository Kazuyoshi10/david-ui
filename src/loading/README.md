# `<Loading>`
Creates an animated `Avatar` component which is displayed while pages/views are loading.

## Props
### name: `string`
The page name used in the loading message.

### error: `string`
Specifiy a custom message to return if there is an error.

## Example
```
<Loading 
  name={ 'page name' }
/>
```
